# CMSC396H Project: Prompt fingerprinting

# File Structure

- submodules: forks of code from papers
- ```results```
    - ```Hui```: output from the code for "PLEAK: Prompt Leaking Attacks against Large Language Model Applications" by Hui et al. and test results from our pipeline
    - ```Pape```: output from the code for "Prompt Obfuscation for Large Language Models" by Pape et al.
- ```run_tests.py```: tests extraction techniques against obfuscation techniques
